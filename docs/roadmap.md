# Roadmap

## Milestone 0 — Project scaffold
- Create Vite React + TS app
- Add TailwindCSS
- Add shadcn/ui baseline
- Add routing

## Milestone 1 — Camera round (no backend)
- Camera permission flow + preview
- Countdown + 10s timer
- Capture frames during the round
- Show “face detected” indicator
- Local-only results screen

## Milestone 2 — Scoring v1
- Face detection + landmarks
- Normalize landmarks
- Compute similarity vs reference representation
- Produce score (0–100) and points
- Basic validation (no face → 0)

## Milestone 3 — Leaderboard v1
- Persist results (local storage first or simple backend)
- Leaderboard UI
- “Your rank” highlighting

## Milestone 4 — Rooms (optional MVP+)
- Create/join room
- Same round start time for all players
- Room leaderboard

## Milestone 5 — Quality
- Better lighting guidance + UX polish
- Anti-cheat improvements
- Performance tuning
- Accessibility pass
